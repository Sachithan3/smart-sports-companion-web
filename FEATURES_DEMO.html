👋 Feature 3: User Greeting & Visit Counter

What's New:

The application now greets users by name and tracks how many times they've visited the page using browser localStorage!

Implementation Details:

// Get or prompt for user name
let userName = localStorage.getItem('userName');
if (!userName) {
  userName = prompt('Please enter your name:');
  localStorage.setItem('userName', userName);
}

// Track and increment visit count
let visitCount = localStorage.getItem('visitCount');
visitCount = visitCount === null ? 1 : Number(visitCount) + 1;
localStorage.setItem('visitCount', visitCount);

// Display personalized greeting
userGreetingDiv.innerHTML = `
  <h2>Hello, ${userName}! 👋</h2>
  <p>You have visited this page ${visitCount} times.</p>
`;

How It Works:

  • ✓ First Visit: Prompts user for their name
  • ✓ Name Storage: Saves name in localStorage (persistent)
  • ✓ Visit Tracking: Increments counter on each page load
  • ✓ Personalized Message: Displays "Hello, [Name]! 👋"
  • ✓ Count Display: Shows visit count with proper grammar
